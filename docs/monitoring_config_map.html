EasyShop Monitoring Config Map

Visualizing the Local Helm Configurations, Dependency Tree, and Kubeadm Manifests

1. Umbrella Chart Structure

  • • Chart.yaml sets up child dependencies: kube-prometheus-stack and loki-stack.
  • • values.yaml holds global chart defaults, overridden dynamically per environment (values-dev.yaml / values-prod.yaml).
  • • Templates are parsed during deployment to inject custom rules and dashboards.

2. Quotas & Routing Rules

  • • namespace-quota.yaml configures strict physical constraints for monitoring, safeguarding host memory and CPU resources.
  • • alertmanager-config.yaml maps alerting triggers to Slack webhooks or email alerts.
  • • rbac.yaml defines ServiceAccounts and RoleBindings required for log and metrics collectors.

3. Scrape & Aggregation Rules

  • • alert-rules.yaml defines thresholds (e.g. CPU > 85%, disk low) to fire alertmanager events.
  • • promtail-config.yaml details scraping jobs, tailing logs from specific filesystem paths and containers.
  • • recording-rules.yaml contains pre-computed expressions for heavy metrics to speed up dashboard loads.
